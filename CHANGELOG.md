# Changelog

## [1.0.0] - 2026-04-20

### Added
- Initial release
- Minimap button with right-click hide option (`/dker minimap` to restore)
- `/dker` slash command to open settings
- Per-spec (Blood, Frost, Unholy) Main Hand and Off Hand rune configuration
- On-screen animated warning when the wrong rune is equipped
- Warning persists until the correct rune is applied
- Configurable trigger: Dungeon, Mythic+, Raid, or Everywhere
